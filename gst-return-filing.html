Frequently Asked Questions

Do I need to file GST return with no sales?

Yes. If GST registration is active, nil return filing may still be required for the applicable period.

What happens if GST returns are delayed?

Late fees, interest, restrictions, and compliance issues may apply depending on delay and return type.
